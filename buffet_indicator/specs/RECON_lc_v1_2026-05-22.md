## 3. Statistical methodology (all SEALED — no implementer choice)

| Topic | Locked choice |
|---|---|
| PIT z-score | Expanding window, sample SD (Bessel n−1), strict PIT excluding current obs, min n=120 |
| Frequency | Month-end-of-month for all components |
| Predictive specification | `r_{t,t+h} = α + β·LC_t + ε`, h ∈ {1Y, 3Y, 5Y, 10Y} |
| Standard errors | Newey-West HAC, lag `L = h·12 − 1` |
| Bias correction | Stambaugh (1999) analytical + bootstrap cross-check |
| OOS R² | Goyal-Welch (2008) prevailing-historical-mean benchmark |
| Nested-model test | Clark-West (2007) MSPE-adjusted |
| Near-unit-root CIs | Campbell-Yogo (2006) when ρ > 0.95 |
| Bootstrap | Stationary block (Politis-Romano 1994), block length `b_opt = ceil(2·N^(1/3))` (Politis-White 2004), 10K reps standard / 50K for tail probs, seed=42 |
| Stationarity | ADF + KPSS + Phillips-Perron + Zivot-Andrews per component + composite |
| Multicollinearity | VIF, Pearson, Spearman, condition number, eigenvalue spectrum |
| Structural breaks | Bai-Perron BIC, max 5 breaks, min regime 30 months, trim 15% |
| Calibration | Brier + Murphy 1973 decomposition, reliability diagram, isotonic overlay, log/CRPS, PIT histogram |
| Backtest split | 1986-01 → 2010-12 estimation / 2011-01+ validation (TIER2/DEEP); 2013-01+ validation (FULL) |

## 8. Falsifiability decision (locked, sealed in `a8635ef`)

| # | Criterion | Threshold | Apply to |
|---|---|---|---|
| 1 | OOS R² (Goyal-Welch) on 1Y SPX TR | > 0.005 | LC_TIER2 |
| 2 | OOS R² on 3Y SPX TR | > 0.020 | LC_TIER2 |
| 3 | OOS R² on 5Y SPX TR | > 0.040 | LC_TIER2 |
| 4 | Predictive β sign | Positive with HAC t > 1.65 (1-sided p<0.05) | LC_FULL |
| 5 | ADF rejects unit root on each component z-score (p < 0.10) | All 5 | Per-component |
| 6 | VIF < 5 cross-component | Max VIF | Per-component panel |
| 7 | Bonferroni-corrected p (20 tests, α/20 = 0.0025) | ≥1 sig | LC_FULL or LC_TIER2 |

Decision: `n_pass ≥ 5` → PASS / `n_pass == 4` → PASS_WITH_CAVEATS / `n_pass ≤ 3` → FAIL. Verdict affects dashboard framing only — pipeline always runs.
