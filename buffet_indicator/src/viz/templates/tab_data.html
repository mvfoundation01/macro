Data — Inspect & Download

Browse the raw outputs powering this dashboard. Download any time series as CSV for offline analysis. All downloads are client-side (no network requests) — they extract data from the JSON payload already embedded in this page.

1. Downloadable CSV exports

One button per dataset. The CSV string is materialized only on click.

z_history: per-variant, per-frame z-score by month. value_history: per-variant raw level by month. sp500_with_regime: S&P 500 monthly close + MVCI regime label. scatter_data: per-variant z-score paired with realized forward returns; this CSV is reconstructed in the browser from the inline chart data on click (v8b.1 bundle-size optimization). For full pipeline output, run the model offline and use the outputs/charts/*.parquet files directly.
